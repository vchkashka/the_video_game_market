# Проект "Рынок видеоигр"

Данный проект был выполнен в ходе обучения в Яндекс.Практикуме профессии "Аналитик данных".

## Данные

В наличии были следующие данные о продажах игр разных жанров и платформ, а также пользовательские и экспертные оценки игр:
* `Name` — название игры.
* `Platform` — название платформы.
* `Year of Release` — год выпуска игры.
* `Genre` — жанр игры.
* `NA sales` — продажи в Северной Америке (в миллионах проданных копий).
* `EU sales` — продажи в Европе (в миллионах проданных копий).
* `JP sales` — продажи в Японии (в миллионах проданных копий).
* `Other sales` — продажи в других странах (в миллионах проданных копий).
* `Critic Score` — оценка критиков (от 0 до 100).
* `User Score` — оценка пользователей (от 0 до 10).
* `Rating` — рейтинг организации ESRB (англ. Entertainment Software Rating Board). Эта ассоциация определяет рейтинг компьютерных игр и присваивает им подходящую возрастную категорию.

## Задача

Сделать предобработку данных, чтобы команда игры "Секреты Темнолесья" смогла изучить развитие игровой индустрии в начале XXI века. Команда хочет выпустить статью на одном известном IT-ресурсе, которая должна привлечь внимание людей, любящих старые игры, и заодно заинтересовать их «Секретами Темнолесья».
